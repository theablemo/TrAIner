# TrAIner: Your AI Squat Coach 🏋️‍♀️🤖

**🗣️ As TrAIner is part of a research program, please report your feedback on [this link](https://urls.st/TrAIner-User-Evaluation-English). This survey will take 5-10 mins of your time.** 

## 📱 About
TrAIner is an Android application developed as a bachelor's thesis project. It uses computer vision and artificial intelligence to help users improve their squatting technique.

## 🔑 Key Features
- 👤 User profile creation
- 📷 Real-time squat pose tracking
- 🔢 Automatic rep counting
- ⚠️ Form error detection
- 💬 LLM-powered chat for personalized advice
- 📊 Session history tracking

## 🛠️ Technical Details
- Platform: Android
- Minimum SDK: 26
- Libraries: Flutter

## 🚀 Getting Started
1. Clone this repository
2. Open the project in Android Studio
3. Build and run the application on an Android device or emulator

## 🎯 Usage
1. Create a user profile
2. Start a new squatting session
3. Position your phone to capture your full body
4. Perform squats and receive real-time feedback
5. After the session, chat with the LLM for additional advice
